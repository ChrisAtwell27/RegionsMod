Installation

On this page

    Server-side Fabric mod. Only the server needs the mod.

    Requirements

    ComponentVersion
    Minecraft1.21.11
    Fabric Loader0.19.2+
    Fabric API0.141.3+1.21.11
    Java21+

    Install

    1. Set up a Fabric server for Minecraft 1.21.11.
    2. Download the RegionsMOBA jar from the Releases page.
    3. Download Fabric API.
    4. Place both jars in mods/.
    5. Start the server.

    Config is created at config/regionsmoba.json on first launch.

    Verify

    Run /regions version in-game.
