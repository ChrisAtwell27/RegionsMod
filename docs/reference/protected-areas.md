# Protected Areas

Any location registered with a `/regions` command is protected.

## Blocked inside protected areas

- Mining blocks
- Placing blocks
- Interacting with block entities (chests, furnaces)
- Damaging entities marked as protected (traders)

## Explicit exceptions

- Mining an ore deposit block is allowed. The block converts to cobblestone and starts a regen timer.
- Breaking the ocean conduit is allowed. Each break removes 1 HP and respawns the block.
- Feeding fuel to the nether furnace is allowed.
- Right-clicking the plains composter with a crop is allowed.

## Unprotected areas

Anywhere not registered is free to mine, place, and interact with normally. This is the default for most of the map.
